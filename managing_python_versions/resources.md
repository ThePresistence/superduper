### SEMANTIC VERSIONING 

**MAJOR.MINOR.PATCH** 

* **MAJOR** -> breaking change 
* **MINOR** -> Adding new functionality, feature, and backward compatible. 
* **PATCH** -> just a bug fix.

Official documentation for the semantic version: [click](https://semver.org/)
